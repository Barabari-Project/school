<b> The bold tag is used to give text emphasis. It makes text inside look thicker and darker

<i> The italic makes text inside look slanted/oblique

<u> The underline tag is used to underline text, a bit like in textbooks for important sentences

<mark> The mark tag is used to highlight text, a bit like in textbooks for important sentences

<img> We use this tag to insert an image into the webpage. The 'src' attribute specifies the source link of the image. The height attribute sets the height of the image to 333 pixels. So, in this case, it displays an image with the provided URL and a height of 333 pixels.
